Приложение для конвертации данных из БД в XML с последующей выкладкой на sftp-сервер

Запускаем: _run.bat

Результат: формируется Xml-файл в папке OUT, затем выкладывается на sftp

Характеристики приложения:
Модульность
Настраиваемость

Причины неполного соответствия выходного XML-файла с образцом:
1) отсутствие сортировки строк
2) Неверно, относительно ТЗ, были заполнены поля Active_KL в образце.
3) Формат csv не поддерживает явное указание типа данных в таблице, поэтому поле Credit_Limit заполняется числом без оговорённого в ТЗ количества знаков после десятичной точки. При работе с MS SQL данные записываются как надо, если они хранятся как строковые или преобразуются к строке в выборке.

Настройки SFTP сервера вынесены в sftpsetup.bat.
Дата и время формируются с точностью до минуты (без секунд, секунды равны нулю).

Выкладка на sftp производится с парольной аутентификацией, без SSL-сертификатов. При необходимости можно прикрутить сертификаты.

Описание файлов:

curl            -- папка с приложением "curl" для связи с sftp
out             -- В эту папку пишутся xml-файлы, которые потом и отправляются на сервер

_run.bat        -- Пакетный файл для запуска приложения
clients.csv     -- таблица с исходными данными
convert.rules   -- Описание правил преобразования для настройки приложения
data.udl        -- Файл с настройками БД
error.log       -- примеры логирования ошибок (очень упрощённо, на бегу делал)
ListSftp.bat    -- пакетный файл для просмотра папки на sftp-сервере. Пожно использовать для проверки соединения.
query.sql       -- запрос к БД. Именно отсюда берутся данные для конвертации.
readme.txt      -- вы здесь :)
sftp_files.txt  -- листинг папки тестового sftp сервера после работы приложения.
sftpsetup.bat   -- Пакетный файл, в который вынесены настройки соединения с сервером
UseDate.bat     -- Пакетный файл дя определения текущих даты и времени. С точностью до минуты.
XmlExport.exe   -- Исполняемый файл приложения конвертации.
